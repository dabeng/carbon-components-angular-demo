By default the carbon grid is 12 in large sizes, 8 in mid sizes, 4 in small sizes.

With columnNumbers you set the size of the column, you have a grid of 12 by default, so if you set for example {{ '{' }} 'lg': 12, 'md': 8, 'sm': 4 {{ '}' }}, that column will get all the available space on that row, if you set two columns with columnNumbers="{{ '{' }} 'lg': 6, 'md': 4, 'sm': 2 {{ '}' }}", each column will have half the available space on that row, etcâ€¦
